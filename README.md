# config
コンフィグアラカルト！！
Ansibleとかでcurlに使ったり、エッジデバイスのキッティング用の自分用スクリプト、設定集。

Grafanaのダッシュボードのパネル一部以外はWTFPLライセンス。
つまり好きに改変して。ライセンス汚染とか設定ファイルに有ると困るので好きにどうぞライセンスです。

詳しくはLICENSEファイルにて。

-----


This repository is a collection of ready-to-use configurations.
You can copy, modify, and apply them directly to your environment.

Think of it as a lightweight, copy-paste friendly alternative to Ansible.
